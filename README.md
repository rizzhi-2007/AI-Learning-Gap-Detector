# AI Learning Gap Detector

An AI-powered web application that helps learners identify gaps in their understanding and receive personalized learning recommendations. The platform analyzes user responses, highlights weak concepts, and provides a structured path to improve knowledge efficiently.

## Features

* 📊 Track learning progress
* 🧠 Identify knowledge gaps
* 🎯 Personalized learning recommendations
* 📈 Interactive dashboard with progress visualization
* 📱 Responsive design for desktop and mobile
* ⚡ Fast and modern user experience

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/learning-compass.git
```

Navigate to the project directory:

```bash
cd learning-compass
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```

## Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
